# 学前先看

## 1. 认识ArmPi Pro

### 1.1产品简介

ArmPi Pro是一款基于ROS操作系统开发的智能视觉搬运机器人，以树莓派5为主控，采用Python编程，在原ArmPi FPV视觉机械臂的基础上，它增加了一个可以全向移动的麦克纳姆轮底盘，使得机器人可以进行移动抓取、目标追踪、智能搬运等功能。

ArmPi Pro底盘可以自由拆卸。它既可以作为桌面视觉机械臂使用，也可以作为移动视觉机械臂使用，是一款高性价比的二合一双模态教育机器人产品。
